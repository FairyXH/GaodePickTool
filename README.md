# GaodePickTool
高德地图HTML本地坐标拾取器

支持搜索、卫星图层切换、坐标自动换算成GCJ-02、BD-09、WGS-84坐标系坐标。

使用方法:下载本仓库的index.html ，在HTML源码中填写 KEY + 安全密钥 ，在浏览器中打开本HTML即可使用。

KEY以及安全密钥获取方法：前往高德地图开放平台注册并认证一个账号 --> 在 控制台,应用管理,我的应用 中创建新应用(任意) --> 为该应用添加Key,服务平台选择Web端(JS API),其他设置均默认 --> 复制KEY和安全密钥并填写于本源码中
